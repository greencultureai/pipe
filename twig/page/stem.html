Stem

The Stem component is a pass-thru proxy for different cannabis regulatory compliance engines. It provides input filtering, generous logging and improved error reporting.

Once you have configured a pass-thru target then you will simply make requests to this configured end-point. 


curl \
	--request POST \
	--header "as-necessary" \
	--data-binary "as-necessary" \
	https://pipe.openthc.com/stem/abcd1234

You must register with this system to get a defined end-point.
